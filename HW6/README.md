## Dijkstra
- 可以用來找某個起點到其他每個點的最小Cost。
- 時間複雜度:O(E+VlogV)

## Kruskal
- 可以用來找一個Graph中把所有點跑完的最短路徑
- 時間複雜度:O(ElogV)

<img src="/pic/Kruskal.png" width="500" height="300">


(E:Edge、V:Vertice)
