## Merge Sort
> 將list從中間切為左數列及右數列</br>
並重複執行直到每個數列中只有一個數</br>
接著再講左右數列內的數兩兩比較大小</br>
比較小的先放入新的數列，直到左右數列都放入。

## Heap Sort
> 將原始的數列以Tree的形式排列</br>
先從下到上將每個小樹的root變為那棵樹中最大的(Max Heap)</br>
再把最上面的數取出，填入最後一個樹，從上到下把每個小樹root變為那棵樹中最大的(Sift-Down)</br>
重複上述動作直到數列排列完成。
